2018年全国大学生英语竞赛哈尔滨理工大学初赛成绩

  • 发布日期：2018-05-02
  • 浏览次数：1767

 

1、 2018年全国大学生英语竞赛初赛于4月15日结束，哈尔滨理工大学（含荣成校区）共有1316名学生参赛，我校C类比赛共有7人进入决赛，34人获省级A类二等奖，61人获省级A类三等奖。B类比赛1人进入决赛、6人获省级A类二等奖、9人获省级A类三等奖。

具体获奖名单详见附表。2018年全国大学生英语竞赛哈尔滨理工大学初赛B类获奖名单.xlsx

                                          2018年全国大学生英语竞赛哈尔滨理工大学初赛C类获奖名单.xlsx

2、决赛时间：决赛笔试（含听力；不含口试）定于2018年5 月20 日（星期日）进行，具体要求将于近期通知参加决赛学生。

 

3、未获奖的参赛学生如查询成绩请到西区新主楼E0203查询。

 
 
                                            
                                                        哈尔滨理工大学教务处 外国语学院
